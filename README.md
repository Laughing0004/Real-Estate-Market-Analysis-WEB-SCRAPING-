# 🏠 Real Estate Rental Market Analysis

## 📌 Project Overview

This project analyzes rental property data across four major Indian cities to understand pricing trends, key influencing factors, and overall market behavior.

Using web scraping, data cleaning, and exploratory data analysis (EDA), raw real estate listings were transformed into meaningful business insights.

---

## 🎯 Business Problem

Rental prices vary significantly across cities and property types.

Key questions addressed in this project:

- Which city has the highest rental prices?
- Does BHK affect rent?
- Does area (sqft) influence pricing?
- Which property type dominates the rental market?

This project answers these questions using data-driven analysis.

---

## 🔍 Major Findings

- Rental prices vary significantly across cities  
- Mumbai represents the premium rental market  
- Area (sqft) has a strong positive impact on rent  
- Higher BHK properties generally have higher rental prices  
- Flats dominate the rental property market  

---

## 📈 Key Insights

✔ Rental market is predominantly residential  
✔ Pricing varies across cities  
✔ BHK and Area are major pricing drivers  
✔ Premium cities command higher price per sqft  

---

## 🧠 Overall Conclusion

Rental pricing is strongly influenced by:

- Location  
- Property Size (Area in sqft)  
- Number of Bedrooms (BHK)  

Using data analysis techniques, raw real estate listings were converted into actionable business insights.
